Verifies that JUnit lifecycle method names match their expected values based on annotations.

This inspection checks methods annotated with `@BeforeAll`, `@BeforeEach`, `@AfterAll`, or `@AfterEach` in JUnit test classes. If a mismatch is found, the issue is flagged to enforce consistent naming conventions for test lifecycle methods across your project.
